# Capítulo 1 – Avaliação de Vulnerabilidades

Este capítulo documenta meu primeiro laboratório de cibersegurança.

Utilizei a ferramenta Nessus para realizar uma varredura de vulnerabilidades
em um ambiente controlado de laboratório com máquinas virtuais.

O objetivo foi compreender:
- o papel de um scanner de vulnerabilidades
- a leitura de resultados (CVSS)
- a diferença entre identificar falhas e explorá-las

Nenhuma exploração foi realizada.
